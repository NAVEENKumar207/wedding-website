# 💖 Tharun & Poongodi — Royal Wedding Celebration Website

An interactive, luxury wedding website designed with modern web aesthetics, smooth 3D animations, custom music player controls, real-time Tamil/English translation, and interactive celebration experiences.

---

## ✨ Key Features

- 🚪 **3D Ceremonial Sliding Door Entrance**: 
  - Features an ornate entrance overlay with dual horizontal sliding door panels, golden light seam flares, rotating sunburst rays, an orbiting gold ring, and a falling floral petal shower.
  - Smooth 3D depth zoom reveal powered by GSAP.

- 🎵 **Interactive Music Player**:
  - Embedded background wedding music player with custom play/pause controls.
  - Floating widget pinned gracefully near closing sections for seamless user interaction.

- 🌐 **Bilingual Support (Tamil & English)**:
  - Real-time language switching with a dedicated circular translate button (`அ` / `A`).
  - Translates all titles, invitations, schedules, venue info, and interactive text effortlessly.

- 💖 **Love Destiny (FLAMES) Calculator**:
  - Interactive couple name evaluator with floating 3D gradient vector hearts and full-screen golden confetti celebrations (`canvas-confetti`).

- 🎟️ **Interactive Canvas Scratch Card**:
  - Interactive scratch-and-reveal card allowing guests to scratch off a gold layer to unveil the wedding invitation.

- 📍 **Venue & Location Details**:
  - Clear breakdown of Wedding (Muhurtham) & Reception events with timings, venue addresses, Google Maps links, and calendar integration.

- 🎨 **Luxury Aesthetics & Design Token System**:
  - Tailored gold palettes (`#d8b65a`, `#b89030`), glassmorphism cards, hanging brass diyas (lamps), lotus corner ornaments, and responsive mobile-first layouts.

---

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Animations**: GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Interactive FX**: Canvas Confetti (`canvas-confetti`), Fireworks.js
- **Typography**: Google Fonts (*Cormorant Garamond*, *Montserrat*, *Great Vibes*)

---

## 🚀 How to Run Locally

1. Clone or download the repository:
   ```bash
   git clone https://github.com/NAVEENKumar207/wedding-website.git
   ```
2. Open `index.html` in any modern web browser or run using a local development server (e.g. Live Server).
